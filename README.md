# LLL - Infrared Light Therapy Website

A professional, modern website showcasing LLL's infrared light therapy products and services.

## 📁 Project Structure

```
lll-website/
├── index.html          # Main website file (all-in-one)
└── README.md           # This file
```

## 🚀 Quick Start

### Option 1: Open in Browser Directly
Simply double-click `index.html` to open it in your default web browser.

### Option 2: Using a Local Server (Recommended)
For better performance and to test all features:

**Using Python 3:**
```bash
cd c:\Users\ADMIN\Downloads\lll-website
python -m http.server 8000
```
Then visit: `http://localhost:8000`

**Using Python 2:**
```bash
cd c:\Users\ADMIN\Downloads\lll-website
python -m SimpleHTTPServer 8000
```

**Using Node.js (http-server):**
```bash
npx http-server
```

## 🎯 Customization Tips

### Change Colors
Edit the color values in the `<style>` section:
- Primary color: `#667eea` (purple)
- Secondary color: `#764ba2` (darker purple)
- Accent color: `#ff6b6b` (red)

### Add Your Contact Info
Update the footer section with your actual:
- Email address
- Social media links (TikTok, Zalo)
- Phone number

### Modify Product Info
Edit the product cards in the products section to update:
- Descriptions
- Wavelengths
- Results
- Target audience
